Основы

Что такое булева алгебра?

Раздел математики, где переменные могут принимать только два значения: истина (1) или ложь (0). Названа в честь математика Джорджа Буля (1854 г.).

Пример из жизни
Свет включён? → 1 (да)
Свет выключен? → 0 (нет)

Дверь закрыта? → 1
Дверь открыта? → 0
Почему важно
Процессор компьютера
выполняет миллиарды
таких операций в секунду.

Всё — только 0 и 1.

Операции

Три базовые операции

AND · Конъюнкция

A ∧ B

"И" — результат 1, только если оба условия верны.

0 ∧ 0 = 0
1 ∧ 0 = 0
1 ∧ 1 = 1

OR · Дизъюнкция

A ∨ B

"ИЛИ" — результат 1, если хотя бы одно условие верно.

0 ∨ 0 = 0
1 ∨ 0 = 1
1 ∨ 1 = 1

NOT · Отрицание

¬A

"НЕ" — инвертирует значение: из 1 делает 0, и наоборот.

¬0 = 1
¬1 = 0

Жизненный пример

Кофе утром: логика в быту

Вы встаёте и думаете: «Выпью ли я кофе?» Переведём это на булеву алгебру.

A = «есть кофе дома»
B = «есть время»

Пью кофе = A ∧ B
(только если есть И кофе, И время)

Беру кофе в кофейне = ¬A ∧ B
(нет кофе дома, но время есть)
Вывод

Наш мозг постоянно решает булевы задачи — математика просто даёт им точный язык.

Производные функции

Другие булевы функции

XOR · Исключающее ИЛИ

A ⊕ B

Равно 1, только если значения разные.

0⊕0=0 · 1⊕0=1 · 1⊕1=0
Пример

«Либо чай, либо кофе — но не оба сразу»

NAND · НЕ-И

¬(A ∧ B)

Инверсия AND. Равно 0 лишь когда оба равны 1.

0·0→1 · 1·0→1 · 1·1→0
Интересный факт

Только из NAND можно собрать любую логику!

NOR · НЕ-ИЛИ

¬(A ∨ B)

Инверсия OR. Равно 1 только когда оба 0.

0·0→1 · 1·0→0 · 1·1→0

XNOR · Эквивалентность

A ↔ B

Равно 1, если значения одинаковые.

0·0→1 · 1·0→0 · 1·1→1

Законы

Основные законы булевой алгебры

Идемпотентность

A ∧ A = A
A ∨ A = A

«Включить включённый свет — он всё равно включён»

Закон де Моргана

¬(A ∧ B) = ¬A ∨ ¬B
¬(A ∨ B) = ¬A ∧ ¬B

Отрицание суммы = произведение отрицаний

Дистрибутивность

A ∧ (B ∨ C) =
(A ∧ B) ∨ (A ∧ C)

Поглощение

A ∨ (A ∧ B) = A
A ∧ (A ∨ B) = A

«Если условие уже включает другое — оно поглощает его»

Применение

Где всё это используется?

Процессоры

Каждый транзистор в чипе — это булев элемент. Миллиарды переключений в секунду.

Поиск

Google: «кофе AND Бишкек NOT дорого» — это дословно булев запрос.

Программирование

if (авторизован AND подписка) — каждое условие в коде это булева функция.

Базы данных

SQL-запросы WHERE используют AND, OR, NOT для фильтрации данных.

Электроника

Светофоры, лифты, сигнализации — вся логика работы на булевых схемах.

ИИ

Нейросети — миллионы логических операций, собранных в граф вычислений.
